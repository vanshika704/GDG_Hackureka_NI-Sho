⚔️ Hackureka 2.0 — Hackathon Event Website

A modern, high-performance hackathon landing platform designed to showcase event details, themes, and enable seamless participant onboarding with an engaging UI experience.

🚀 Overview

Hackureka 2.0 is a national-level hackathon website built to provide participants with all essential information in a clear, interactive, and visually appealing format.

It focuses on:

Structured event presentation
Smooth navigation and user flow
High engagement through animations
✨ Features
🎯 Hero section with event highlights and call-to-action
📄 Detailed sections (About, Themes, Timeline, FAQs)
📱 Fully responsive design (mobile + desktop)
⚡ Smooth animations using Framer Motion
🔗 Integrated registration and external event links
🧭 Clear navigation and content hierarchy
🧠 Key Highlights
Built with component-based architecture for scalability
Focused on UI/UX and visual hierarchy
Designed to maximize user engagement and conversions
Optimized for performance and fast load times
🛠 Tech Stack

Frontend

React
TypeScript
Tailwind CSS
Framer Motion
🏗 Architecture
Modular React components for each section
Utility-first styling using Tailwind CSS
Animation layer powered by Framer Motion
Clean separation of UI and logic

🎯 Purpose

This project aims to:

Provide a centralized platform for hackathon information
Improve participant onboarding experience
